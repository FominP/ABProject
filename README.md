# 📊 AB Test Analyzer & Calculator

> Пет-проект: простая Python-библиотека для статистического анализа A/B-тестов и веб-калькулятор на её основе.

## 🧠 Зачем это?

Я решил разобраться в A/B-тестировании «снизу вверх»:  
– написать собственную реализацию основных статистических критериев;  
– завернуть их в удобную библиотеку;  
– сделать веб-интерфейс, чтобы можно было быстро считать p-value не открывая IDE.

Это учебный проект, поэтому код местами может быть упрощён, но все расчёты проходят проверку на классических примерах.

## ✨ Возможности (сейчас и в планах)

### ✅ Уже реализовано в библиотеке
- **Z-тест для пропорций** (двухвыборочный)
- T-тест для средних (равные/неравные дисперсии) – *в разработке*
- Расчёт необходимого размера выборки (power analysis)

### 🌐 Онлайн-калькулятор
Доступен по ссылке: [ваш_логин.pythonanywhere.com](https://ваш_логин.pythonanywhere.com)  
Выберите тест, введите данные – получите p-value и заключение.

## 🛠️ Установка и использование

### Локальная установка библиотеки
```bash
git clone https://github.com/FominP/ABProject.git
cd ab_testing
pip install ./ab_test_analyzer
```

Запуск калькулятора (локально)
```bash
cd calculator
pip install -r ../requirements.txt   # flask, numpy, scipy
python app.py
```
Затем откройте http://127.0.0.1:5000

Использование библиотеки напрямую
```python
from ab_test_analyzer.core import ztest_proportions

result = ztest_proportions(conv1=200, n1=1000, conv2=250, n2=1000)
print(result['p_value'])   # ≈ 0.005
```
## 🚀 Деплой (как работает онлайн-версия)
Калькулятор запущен на PythonAnywhere (бесплатный тариф).
При повторном запросе первый запуск может быть чуть медленнее (холодный старт).
